    各授業回の学習事項

    第1回WWWとHTML

    第1回では事前課題の確認を軽くした後HTMLを使い文書の作成をしました。HTMLでは文書を作成するのに要素で作られます。!DOCTYPE html、html、head、bodyが固定で入っていて抜くことができません。また要素では決められた名前だったり。また、少し間違っていても実行できてしまうことがあるので注意が必要。

    演習の成果：演習1-4:HTMLのさまざまな要素

    第2回CSSによるWedページの装飾

    第2回ではCSSを使いウェブページに彩りや枠などをつけられるようにしました。 SCCは外部スタイルシートでHTMLと組み合わせ、別ファイルで作成する。 またプロパティ名は様々あり、CSS においてあらかじめ定められた名前だけが使用でき、 プロパティ値については，プロパティ名に対して許された値だけを指定すると使えます。

    演習の成果：演習2-3:Webブラウザの紹介

    第3回JavaScriptの基本

    第3回ではJavaScript は Web ページ上で複雑な機能を実現するプログラミング言語で、 Web ページを開いた直後や，ボタンを押したときに JavaScript プログラムが実行する。 ページの自動更新やアニメーションなど動きのあるページを実現するために JavaScript プログラムを使う。またその場でプログラムを試したりもできる。

    演習の成果：演習3-6:javascript
