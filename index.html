    数当てゲーム

    数当てゲームの概要としては読み込みをした時、 １から１０までの整数の中でランダムな数字を答えとして生成しユーザーが解答ボックスに整数を打ち込んだ後、 解答ボタンを押してもらうと何回目の予想かと、 正しいか違うかどうかと間違っていた時はヒントを解答ボックスの下に出してくれます。 また3回間違えを繰り返してしまうと答えを表示してくれます。 そして4回目以降も解答ボタンを押して予想すると答えとともに「すでにゲームは終わっています」 とアナウンスをさせていただきます。 もう一度最初からやり直すときは再読み込みをしてもらうと答えを生成し直してくれて新しくゲームに挑戦することができます。 プログラムとしてはaddEventListener('click')でボタンを押した時のイベント処理を行います。 また.valueで入力された文字を文字列として読み取ります。

    各授業回の学習事項

    第1回WWWとHTML

    第1回では事前課題の確認を軽くした後HTMLを使い文書の作成をしました。HTMLでは文書を作成するのに要素で作られます。!DOCTYPE html、html、head、bodyが固定で入っていて抜くことができません。また要素では決められた名前だったり。また、少し間違っていても実行できてしまうことがあるので注意が必要。

    演習の成果：演習1-4:HTMLのさまざまな要素

    第2回CSSによるWedページの装飾

    第2回ではCSSを使いウェブページに彩りや枠などをつけられるようにしました。 SCCは外部スタイルシートでHTMLと組み合わせ、別ファイルで作成する。 またプロパティ名は様々あり、CSS においてあらかじめ定められた名前だけが使用でき、 プロパティ値については，プロパティ名に対して許された値だけを指定すると使えます。

    演習の成果：演習2-3:Webブラウザの紹介

    第3回JavaScriptの基本

    第3回ではJavaScript は Web ページ上で複雑な機能を実現するプログラミング言語で、 Web ページを開いた直後や，ボタンを押したときに JavaScript プログラムが実行する。 ページの自動更新やアニメーションなど動きのあるページを実現するために JavaScript プログラムを使う。またその場でプログラムを試したりもできる。

    演習の成果：演習3-6:javascript

    第4回DOMとJavaScript

    第4回ではテキストボックスとボタンをページ上に設置し、 ボタンを押したらその内容を処理するためのプログラムを学んだ。 こういう行動はイベント処理と言われていてHTMLとjavascriptを両方使い 連携プレーみたいなことをすることです。 またDOMとはページに設置されているHTML要素の検索と テキスト変更をJavaScriptでできるようにしている。

    演習の成果：演習4-10:計算

    第5回イベント処理

    第5回ではDOM を使って次のことを行う要素の新規作成、 要素の編集、要素の削除の３つを行う。 第4回の時にやった検索をした後insertAdjacentElementなどを使い 要素の編集・再設定などを行なっていき場面によて表示するものを変えること ができるようになりました。

    演習の成果：演習5-7:DOMプログラミング
